# VolunteerHub

VolunteerHub is a web application that connects **motivated volunteers** with NGOs and social initiatives, making it easier to discover and apply for meaningful opportunities in their city.

## Live Demo

- Production: https://volunteerhub-eta.vercel.app/

## Features

- Browse curated volunteering opportunities created by NGOs.
- Email/password authentication for volunteers and organizations.
- Create and manage NGO profiles and opportunity listings.
- Basic dashboards for viewing opportunities and applications.
- Responsive layout suitable for laptops and smaller screens.

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm / yarn / pnpm / bun (any one of them)

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd volunteerhub

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
3. Create a `.env.local` file in the project root and add your environment variables (database, auth, API keys, etc.).

4. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
5. Open `http://localhost:3000` in your browser to view the app.
